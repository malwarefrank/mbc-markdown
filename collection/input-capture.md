Input Capture
=============
Malware captures user input.

**See ATT&CK:** [**Input Capture**](https://attack.mitre.org/techniques/T1056/).

Methods
-------
* **Mouse Events**: Mouse events are captured.
* **Keyboard Events**: Keyboard events are captured.
